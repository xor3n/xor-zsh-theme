# Installation
If you use the oh-my-zsh zsh framework, you can simply run `git clone https://github.com/xor3n/xor-zsh-theme.git; cd xor-zsh-theme; cp xor.zsh-theme /home/$USERS/.oh-my-zsh/custom/themes/`.
